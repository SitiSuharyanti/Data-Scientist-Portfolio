**Skills:** Data Cleaning, EDA, Feature Engineering, Target Encoding, Correlation-Based Feature Selection, Classification, SMOTE, Class Weighting, Threshold Optimization, Model Evaluation (AUC, KS Statistic, Recall, F1, FNR), Gradient Boosting, Random Forest, Logistic Regression
